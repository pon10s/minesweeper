# 🤖 マインスイーパー自動攻略ボット

このマインスイーパーを、**人間と同じ画面操作（タップ＝開く／長押し＝旗）で自動で解くロボット**です。
ゲーム本体（`src/`）には一切手を加えていません。外から画面を読んで操作する「別プログラム」です。

- 確実に分かる手（基本ルール → 定石＝部分集合の法則）を処理
- 確実な手が尽きたら「地雷の確率が一番低いマス」を推測で開く（当たって砕けろ）
- 地雷を踏んだら自動でリセットして再挑戦（勝つまで）
- **クリアしたら名前登録の一歩手前で停止**（ランキングに載せるかは自分で判断。ボットは何も送信しません）

> ⚖️ 自分のゲームを自動で解くのは問題ありません。ダメなのは「ボットの記録を“人間の記録”と偽って
> 共有ランキングに出すこと」。本ボットは登録の手前で止まるので、載せるかはあなたが判断できます。

## ① コンソール版（いますぐ使える）

### 使い方
1. マインスイーパーを開く → https://pon10s.github.io/minesweeper/
2. 難易度（初級／中級／上級）を選ぶ
3. ブラウザの**開発者ツール**を開く
   - PC Chrome/Edge：`F12` → 「Console（コンソール）」タブ
   - Mac Safari：環境設定で「開発」メニューを有効化 → `Cmd+Option+C`
4. 下のどちらかを実行：

**A. かんたん（1行を貼ってEnter）** — 公開中のボットを読み込んで自動スタート：
```js
var s=document.createElement('script');s.src='https://pon10s.github.io/minesweeper/bot/console-bot.js?'+Date.now();document.head.appendChild(s);
```

**B. 直接貼り付け** — `console-bot.js` の中身を全部コピーしてコンソールに貼り、Enter。

5. ロボットが解き始めます。**止めるには** `msBot.stop()` と打ってEnter。

### 動きのメモ
- クリアすると「ベスト10入りおめでとう！」の登録画面で止まります（ボットは名前を入れません）。
- 上級は推測が多く、何度も再挑戦してから勝つことがあります（運ゲー区間があるため）。
- スマホのSafariでもコンソールは使えますが、PCのほうが手軽です。
