# マインスイーパー（スマホ対応・クラシック風）

Windows のクラシックなマインスイーパー（Win95/98 風）を模倣した、スマホで遊べるマインスイーパーです。
バニラ HTML / CSS / JavaScript（ビルド不要）。

## 🎮 遊ぶ

**https://pon10s.github.io/minesweeper/**

## 操作

- **タップ**＝マスを開く（旗マスをタップすると旗を外す）
- **長押し**＝旗を立てる／外す
- **💡ヒント**＝安全マスを1つ光らせる（最後の操作から5秒で押せる）
- **難易度**＝初級(9×9/10)・中級(16×16/40)・上級(16×30/99)。上級はスマホ向けに縦長表示
- **🏆ランキング**＝ベスト10入りタイムを名前付きで共有（ヒント未使用のクリアが対象）

## 記録

- ベストタイム・名前：端末内（localStorage）に保存
- ランキング：Supabase に保存して全員で共有

## 構成

```
tests/            … ロジックの簡易ユニットテスト（tests/test.html）

```
